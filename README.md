# coderwhy vue2 mall 项目(移动端)

## 线上部署(移动端)

- http://124.222.44.115:8080

## 本地预览和开发

```
mkdir mall
git clone git@github.com:Hacker-C/mall.git mall
cd mall
npm install
npm run serve
```

## 打包上线

```
npm run build
```

## 技术栈

- vue2
- vue-router3
- vuex3
- axios
- vue-scroller
- less
- 移动端适配：amfe-flexible + postcss-pxtorem
